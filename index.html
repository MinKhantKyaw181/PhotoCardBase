📵 Camera ကို ဖွင့်ခွင့်မရပါ

Camera permission လိုအပ်ပါတယ်။

  1. Chrome address bar ဘေးက 🔒 (padlock) ကို နှိပ်ပါ
  2. Permissions → Camera → Allow ရွေးပါ
  3. Camera option မတွေ့ရင်: Android Settings → Apps → Chrome → Permissions → Camera ကို Allow လုပ်ပါ
  4. ပြီးရင် ဒီစာမျက်နှာကို reload (ပြန်ဖွင့်) လုပ်ပါ
